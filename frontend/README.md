# LotusTime Frontend

Giao diện người dùng cho hệ thống LotusTime, xây dựng bằng **React** + **Vite**.

## Công nghệ sử dụng
- React 18
- Vite
- React Router DOM
- Phosphor Icons (Light weight)
- Dnd-kit (Dự kiến cho tính năng Kéo Thả TKB)
- CSS Module / Vanilla CSS với CSS Variables (`index.css` & `design-tokens.css`)

## Hướng dẫn phát triển
```bash
npm install
npm run dev
```

Server sẽ chạy ở cổng mặc định `5173`. Giao diện tương tác với API tại `http://localhost:5000`.
